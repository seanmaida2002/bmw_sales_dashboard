# BMW Global Automotive Sales Dashboard
## Objective
Analyze global automotive sales performance and trends using SQL, Excel, and Tableau.

Dataset link: [BMW Automotive Sales Dataset](https://www.kaggle.com/datasets/dmahajanbe23/bmw-global-automotive-sales)

## Step 1: Database Setup & Data Ingestion (MySQL)
I started the project by setting up a relational database environment in MySQL.
- Created a dedicated database for the project
- Designed and implemented a structured table schema based on the CSV dataset
- Defined appropriate data types
- Imported the sales dataset from the CSV into the MySQL table
- Verified successful data load and row counts
